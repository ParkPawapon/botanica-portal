# Changelog

## v0.0.1 - 2026-05-20

Initial foundation release for Botanica Portal.

This release establishes the production-ready baseline for future educational botanical garden development. It includes the Next.js App Router foundation, TypeScript strict configuration, Tailwind CSS, Material UI with Emotion, centralized design tokens, clean architecture folders, placeholder routes, SEO scaffolding, CI quality gates, security scanning, repository governance, and container package publishing.

No end-user botanical content or complete feature pages are included in this version.
